## Sobre o Projeto
O **Hermen@uta** é um laboratório focado na síntese entre o pensamento clássico e as ferramentas do futuro. Aqui, a **Filosofia Medieval (Tomismo)** encontra o **Jornalismo de Tecnologia**, a **IA** e o **Bitcoin**.
